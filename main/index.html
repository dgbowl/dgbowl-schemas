This repository contains the versioned archive of the schemas for the following tools:

  • The Recipe schema for dgpost, with its latest implementation in recipe;

  • The DataSchema schema for yadg, with its latest implementation in dataschema;

  • The Payload schema for tomato, with its latest implementation in payload.
